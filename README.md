Mini Institutional BI Environment – TUS
Overview

This project demonstrates a prototype institutional Business Intelligence (BI) environment for analysing student progression and retention at Technological University of the Shannon (TUS).

The objective is to simulate a warehouse-first, governed KPI framework aligned with higher education analytics best practices.

Project Components

Synthetic institutional data generation (Python)

DuckDB warehouse

dbt transformation & testing

Star schema modelling

Executive-ready Tableau dashboards

Governance and KPI documentation
